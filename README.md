# Sales Receivable Analysis Application

销售与实际应收款分析系统 — 基于 Streamlit + Pandas + Plotly 构建。

## 功能特性

- **销售总览**：KPI 指标、月度趋势、分类分析、客户/产品 TOP 排名、出口国家分布
- **客户分析**：客户画像、回款率、单客户趋势、常购产品分析
- **回款分析**：收款汇总、月度趋势、未回款客户识别、回款率预警
- **应收账款**：应收余额计算、账龄/逾期风险分析、客户对账单查询
- **产品分析（整体与单一客户）**：SKU 排名、产品分类分析、单价波动、色号/等级分析
- **导出报表**：财务汇总、客户对账单、销售/收款明细查询与导出

## 数据上传说明

系统需要手动上传以下数据文件：

### 1. 销售数据 CSV/XLSX

| 字段 | 说明 |
|---|---|
| 单据日期 | 销售日期，支持 "2025年3月1日" 格式 |
| 单据编号 | 销售单号（唯一值） |
| 客户分类 | 开票公账（1账户）/ 现金美金（2账户）/ 出口美金（3账户） |
| 客户名称 | 客户名称 |
| 商品名称 | 产品名称 |
| 色号 | 产品色号 |
| 规格型号 | 产品规格 |
| 类别 | 产品类别 |
| 等级 | 产品等级 |
| 箱数 | 销售箱数 |
| 平方数 | 销售面积 |
| 单价$ | 单价（美金） |
| 合计$ | 美金金额 |
| 汇率 | 汇率 |
| 苏姆合计 | 苏姆金额 |
| 出口国 | 出口目的国 |
| 合同编号 | 合同编号 |

> 允许负数（退货）和空值（赠送）情况。

### 2. 收款数据 CSV/XLSX

| 字段 | 说明 |
|---|---|
| 日期 | 收款日期 |
| 账户类型 | 同销售数据的客户分类：1账户 / 2账户 / 3账户 |
| 客户名称 | 收款客户（与销售数据一致） |
| 美金金额$ | 美金到账 |
| 苏姆金额som | 苏姆到账 |
| 备注 | 收款说明 |

> 允许负数（退款）情况。

### 3. 客户主数据 CSV/XLSX（可选）

来自 contract_customers.xlsx，包含客户名称标准化映射。

| 字段 | 说明 |
|---|---|
| 类别 | 客户归类 |
| 税号 | 客户税号（唯一值） |
| 客户名称 | 标准客户名称（与销售/收款数据对应） |
| 状态 | 可用状态 |

## 安装

```bash
cd sales_receivable_app
pip install -r requirements.txt
```

## 运行

```bash
cd sales_receivable_app
streamlit run app.py
```
